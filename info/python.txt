### Python Environments ################################################################################################

Python relies on a library (aka package) system for its proper funtionality and extended application.

For that purpose there are package managers that help you organise and install these packages.

In practical terms, python is often listed as a package, and installed as such.

There are various managers, but here I only show Conda (the only one I ever used).

Conda can be installed from their website: https://www.anaconda.com/download
(miniconda is essentially a lighter conda in terms of installation, same as conda otherwise and you won't tell
 the difference after installation)
(You might have to sell your soul during installation, press q to exit if stuck)

Once you have installed conda you can activate conda environments with the following command:

>>> conda activate env_name

The default environment is the "base" or "miniconda3" environments, and you should avoid using it for your projects.

If you agreed to it, the default environment will be activated in all your new shells by default.

You can see the name of the current active environment next you your name in the terminal line.

Each environment will have its own python and packages, and will not be shared with other environments.

This means that you can have different environments with different python and package versions without conflict.

You can view the list of available environments by running the command:

>>> conda list env

When starting a python project you should create it's corresponding environment to install packages without conflicting
 with those used in other projects. You can use the following command for that:

>>> conda create -n env_name

!! If using an existing env_name it will erase it and reset it (might be usefull if intended)

To delete environments just remove the respective folder, no extra action needed.

Once created a new environment (and activating it), you can start by installing python in it with the following command:

>>> conda install python

Or if you need a specific version:

>>> conda install python=3.XX
